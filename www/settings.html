Settings

What does the QR code scanner do?

For clarification, this feature is not related to the remote installation process seen in the Installation Guide.

The "Scan QR Code" option in the NetPass settings will activate the outer camera on your 3DS (with the inner camera being an option as well.)

This QR code scanner is used for:

  • Logging into the NetPass website for the first time. This connects your console to your account which allows you to receive donation perks. You can set up alternative login options via Reddit, Gitlab or Discord after that.
  • Receiving specific StreetPasses. Note that this will be a one-sided StreetPass download, so the QR code creator won't receive your StreetPass. A guide to create such QR codes yourself to spread your StreetPass data is provided further down.
  • Joining time-limited event locations. From time to time special event locations will be accessible via a QR code, generally offering more StreetPasses than regular locations. There's no schedule for when these locations will appear, or where the QR codes will be shared. Please refrain from asking/demanding for them.

Create a QR code to download a specific StreetPass

  1. Install the CEC Tool (See Installation Guide for detailed install instructions (but for NetPass))
    • Universal Updater: Search for "CEC Tool"
    • FBI: Scan the QR code for remote installation
    • Manual: Use the latest CEC Tool release
  2. While in CEC Tool, select a Game → Outbox → ID → Export to file
  3. Grab the ID.message.cec file from SD:\cectool\out\ and upload it to a website with "Let's Encrypt" HTTPS certification
    • Catbox works as an example
  4. Grab the URL and use this QR generator specifically
    • The URL needs to directly download the file when opened, it doesn't work if it leads to a page with a download button!
  5. Scan the QR code with NetPass

What is the "Integrations" button for? What happens if I turn the Pretendo one on/off?

The Integrations option you see under your settings is meant for configuring 3rd party services. At the moment, NetPass only offers a Pretendo integration that adds compatibility to Pretendo's SPR.

With the Pretendo integration, NetPass is able to download passes from your Pretendo friends list and distribute them as if they came from NetPass. Otherwise, with this integration toggled off, NetPass SPR will block Pretendo SPR passes.

As a final disclaimer, third party integrations are outside of NetPass moderation and should be used at your own risk.

Custom NetPass Backgrounds

  1. Prepare location images
    • 400x240 pixels
    • .png .jpg or .jpeg file extension
    • Named after a location: "home", "train_station", "plaza", "mall", "beach", "arcade", "cat_cafe", "back_alley", "event_location"
  2. Place images in SD:\config\netpass\background\ (create the folder if it doesn't exist)
