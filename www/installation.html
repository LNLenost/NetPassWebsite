Requirements

Before installing NetPass, you will need:

  • 3DS with system software 11.15 or higher (11.17 recommended)
  • Latest Luma3DS version (If you haven't modded your 3DS yet, please follow this guide.)
    If you are unsure about your installed versions, check this guide for Luma3DS and Section II of this one for system software.

Installation Guide

Via Universal Updater (Recommended)

  • You will need:
    • 3DS with custom Firmware
    • Universal Updater
    • Internet connection
  1. Launch Universal Updater
  2. Search for NetPass and download either the 3DSX or the CIA
  3. The app should then appear on the HOME Menu if using CIA, or in the Homebrew Launcher if using 3DSX

Via FBI (QR Code, CIA only)

  • You will need:
    • 3DS with custom Firmware
    • FBI
    • Internet connection
  1. Launch FBI and select "Remote Install", then select "QR Code"
  2. Point the 3DS camera to the QR Code shown above
  3. Once the camera recognizes the QR, it should install. Press "Ok" and you’re done!

Manual Installation

Only do this if you can't install through Universal and FBI.

  • You will need:
    • 3DS with custom Firmware
    • FBI or GodMode9
    • The latest NetPass release (Click here to download)

Installing Patches

NetPass comes with a few patches that do several different things, such as:

  • Reviving SpotPass
  • Reviving Image Share
  • Enabling NetPass StreetPass Relay (SPR)
    • This downloads StreetPasses in the background!
  • Allowing you to receive Swapnotes from everyone
    • Usually this is only possible between 3DS friends

Installing and enabling the patches

  1. In NetPass go into Settings → Miscellaneous → Update patches → Install
  2. Turn off your 3DS
  3. While holding Select, restart your 3DS
  4. In the Luma3DS configuration menu turn on Enable loading external FIRMs and modules
  5. Save and exit

Updating

If you are on NetPass 1.1.1 or higher, you can update from the app itself! You will be prompted to update to the newest version available whenever you launch Netpass on an outdated version.

On NetPass versions prior to 1.1.1, or if you're having trouble with the in-app updater, you can install newer versions over the old one via the same installation methods from earlier; no need to uninstall. Remember to get the .CIA version for the Home Menu, and/or the .3DSX version if you want to launch from within the Homebrew Launcher.

Uninstalling

  1. Make sure you are using NetPass v1.0.1 or newer.
    • If your NetPass version is older, please update in order to simplify the uninstallation process.
  2. In NetPass go into Settings → Integrations and make sure Pretendo is turned OFF
  3. In NetPass go into Settings → Miscellaneous → Update patches → Remove to uninstall all applied patches.
  4. Wait for your 3DS to restart, then remove NetPass like any other homebrew app.
