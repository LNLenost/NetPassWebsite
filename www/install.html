Requirements

Before installing NetPass, you will need:

  • 3DS with system software 11.15 or higher (11.17 recommended)
  • Latest Luma3DS version (If you haven't modded your 3DS yet, please follow this guide.)
    If you are unsure about your installed versions, check this guide for Luma3DS and Section II of this one for system software.

Installation Guide

Via Universal Updater (Recommended)

  1. Launch Universal Updater
  2. Search for NetPass and download either the 3DSX or the CIA
  3. The app should then appear on the HOME Menu if using CIA, or in the Homebrew Launcher if using 3DSX
Via FBI (QR Code, CIA only)
  1. Launch FBI and select "Remote Install", then select "QR Code"
  2. Point the 3DS camera to the QR Code shown above
  3. Once the camera recognizes the QR, it should install. Press "Ok" and you’re done!
Manual Installation

Only do this if you can't install through Universal and FBI.

  1. Download the latest NetPass release(s) shown above (.cia for HOME Menu apps, .3dsx for Homebrew Launcher apps)
  2. Follow the instructions on the wiki link below.
  3. Installing Software (wiki.hacks.guide)

Uninstalling

  1. Make sure you are using NetPass v1.0.1 or newer.
    • If your NetPass version is older, please update in order to simplify the uninstallation process.
  2. In NetPass go into Settings → Miscellaneous → Update patches → Remove to uninstall all applied patches.
  3. Wait for your 3DS to restart, then remove NetPass like any other homebrew app.
