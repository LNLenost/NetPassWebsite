Known Errors

“libCURL error 6: Couldn't resolve host name”

Make sure wireless is turned on and that you’re connected to a Wi-Fi signal; wait until the connection is confirmed to be working before entering the app (the internet icon should be blue).

“libCURL error 28: Timeout was reached”

Try opening NetPass again, or try a different network; yours might be unstable/too slow. This error could also mean that the server is down; if that’s the case, try again later.

“libCURL error 60: Make sure that your system’s date and time are set correctly!”

Make sure Date and Time on your console are correct

  • Open the Rosalina menu (L + DPAD Down + SELECT)
  • Go to Miscellaneous options
  • Select 'Update time and date via NTP' (You need an internet connection to do this)
  • Use DPAD Left/Right to change your hour offset
  • Press A to update the time and date of your 3DS
  • Press B to return to the Miscellaneous options menu
  • Select 'Nullify user time offset'
  • Restart your 3DS

“MusicNotInitiated (19) - The music module has not been initialised yet.”

Follow Section III of this guide.

“NotAgreeEula (108) - User has not agreed to the EULA”

Use Dead Simple EULA Set.

“HTTP 403: Forbidden”

The server is refusing the request from your console.

You are most likely banned. Please use a recent version of NetPass in order to see the infraction, time of ban and time of unban on the top screen. If you do not get a ban message, or have questions surrounding your ban, please open a ticket our Discord's Mod Mail channel.

Using an emulator can also cause this error, see General Questions .

“409: Failed to enter location”

Either the Netpass on your console is outdated, or you don't have StreetPass enabled on any game. Make sure you install the latest version, and have at least 1 game with StreetPass enabled.

If you get an error that is not listed here:

Please ask for help in our Discord's support channels.

Known Issues

Not receiving passes for a specific title anymore

Please check the following:

  • Is StreetPass enabled in that title?
    • Check System Settings → Data Management → StreetPass Management
    • Check NetPass → Settings → Toggle Games
  • Do you already have a full queue in that title? (Check Home Menu → Notifications)
    • NetPass can only download more StreetPasses once you make room by using the already present ones

I cannot send SwapNote letters to my friends

That feature depends on SpotPass, which got discontinued by Nintendo. NetPass only works with StreetPass.
